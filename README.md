<h3 align="left">🚀 Projects & Contributions</h3>

#### 🔌 **BLE Mesh Gradient Routing (nRF54)**
- Advanced routing implementation for Bluetooth Mesh networks on nRF54 platform
- Focus on optimizing network performance with gradient-based routing algorithms
- **Tech Stack:** C, Bluetooth Mesh, nRF54

#### 🚗 **CAN Register-level for Car Diagnostic**
- Bare-metal firmware project for S32K144 microcontroller
- Implements CAN bus communication for automotive diagnostics
- Manual Makefile setup with GCC toolchain (no IDE required)
- Features: FlexCAN driver, NVM, DTC, and UDS (Unified Diagnostic Services)
- **Tech Stack:** C, ARM Cortex-M4, CAN Protocol, Makefile
- **Repo:** [CAN-Register_level-for-car_diagnostic](https://github.com/Acakua/CAN-Register_level-for-car_diagnostic)

#### 🌐 **Gateway for nRF54L15**
- IoT gateway development for nRF54L15 wireless SoC
- Bridges between BLE devices and cloud/network infrastructure
- **Tech Stack:** C, nRF54L15, Wireless Communication

#### 📚 **Library Management System**
- Full-stack application for managing library operations
- Includes use case modeling and system design
- **Tech Stack:** Software Architecture, Use Case Diagrams

#### 📱 **Android nRF Mesh Library**
- Comprehensive Android library for Bluetooth Mesh provisioning and communication
- Supports all Bluetooth Mesh Profile 1.0.1 features
- **Features:**
  - Device provisioning with OOB authentication
  - Network and Application Key management
  - Secure Network beacons parsing
  - Group management and virtual addresses
  - Generic OnOff and vendor model support
  - Proxy Filter and IV Index updates
- **Tech Stack:** Java, Android, Bluetooth Mesh, Maven Central
- **Repo:** [Android-nRF-Mesh-Library](https://github.com/Acakua/Android-nRF-Mesh-Library)
